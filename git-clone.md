🤖 Machine Learning & AI Architectures 
These repositories help you build intelligent agent frameworks, decision-making pipelines, and custom spatial awareness directly inside Unity. 

• Unity ML-Agents Toolkit: The official repository by  Unity Technologies  for training intelligent agents using deep reinforcement learning via Python. 
• Unity-MCP: A cutting-edge repository providing Model Context Protocol (MCP) toolkits to interface LLM agents (like Claude or GitHub Copilot) directly with the Unity Editor runtime. 
• Fluid Behavior Tree: A pure-code, highly modular behavior tree micro-framework designed specifically for complex actor decisions in Unity. 
• Unity Movement AI: A comprehensive library packed with common steering behaviors, obstacle avoidance, and path-following algorithms. 
• GOAP (Goal-Oriented Action Planning): A multi-threaded, highly optimized GOAP system designed to let non-player characters creatively plan multi-step goals. 
• NPBehave: An event-driven Behavior Tree framework for C# and Unity, optimized to minimize heavy frame-by-frame polling overhead. [11, 14]  

⚙️ Core Engine Systems & Performance Optimization 
If you are rewriting Unity's underlying behavior to make it operate more like a custom, high-frequency data engine, utilize these foundational architectures: 

• Unity C# Reference Source: The official reference code for the Unity Editor and runtime. Essential for understanding how the native engine functions under the hood. 
• ECS & DOTS Samples: The central repository for utilizing the Entity Component System (ECS), C# Job System, and Burst Compiler to maximize CPU multithreading. 
• Netcode for GameObjects: Unity’s high-level networking SDK built to manage core multiplayer engine pipelines synchronously. 
• Prowl Game Engine: A standalone, open-source C# 3D game engine explicitly inspired by Unity's structural design. It is an excellent educational repository for seeing how a "Unity-like" architecture is written from scratch. [18, 19]
